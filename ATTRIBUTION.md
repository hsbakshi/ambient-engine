## Adding More Audio

When adding audio files, please:

1. Ensure the audio is CC0 Public Domain, Creative Commons licensed (with proper attribution), or original
2. Document the source in this file
3. Update the `AUDIO_ATTRIBUTION` object in the soundscape config if needed
4. Keep file sizes reasonable (2-5MB per asset)

### Format for New Entries

```
### [Sound Description]
- **File**: `filename.mp3`
- **Source**: [Source Name](URL)
- **License**: [License Type]
- **Description**: [Brief description]
```
